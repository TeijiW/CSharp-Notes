# Estruturas de Controle

-   ```c#
    if(<condicao>){
        // code
    }
    ```
-   ```c#
    if(<condicao>){
        // code
    }else{
        // code
    }
    ```
-   ```c#
    if(<condicao>){
        // code
    }else if(<condicao>){
        // code
    }
    ```
-   ```c#
      switch(<variavel>){
          case <condicao1>:
            // code
            break;
          case <condicao2>:
            // code
            break;
          default:
            <defaultCode>
            break;
      }
    ```
-   ```c#
      while(<condicao>){
          // code
      }
    ```
-   ```c#
      do{
          // code
      }while(<condicao>)
    ```
-   ```c#
      for(<declaracao>;<condicao>;<incremento>){
          // code
      }
    ```
-   ```c#
      foreach(<tipo> <variavel> in <array>){
          // code
      }
    ```
